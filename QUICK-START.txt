STEP 3: EXPLORE THE FEATURES
=============================

FOR PATIENTS:
1. Click "Find Doctors" to browse doctors
2. Click on a doctor to see their availability
3. Select a time slot and book an appointment
4. Choose payment method (Cash or Card)
5. View "My Appointments" to see upcoming bookings
6. Check "History" to rate completed appointments


FOR DOCTORS:
1. View your dashboard for today's statistics
2. Click "Today's Schedule" to see appointments
3. Mark appointments as done after consultation
4. Go to "Manage Availability" to add time slots
5. View "All Appointments" to see history


FOR ADMINS:
1. View dashboard for system overview
2. Click "Manage Doctors" to add/edit doctors
3. Click "Add New Doctor" to create doctor accounts
4. View "All Appointments" to see system activity
5. Use "Manage Schedules" to control doctor availability

TIPS:
=====
- All data is saved in your browser (localStorage)
- Refresh the page to see updated data
- Clear browser data to reset the system
- Works best on Chrome, Firefox, or Edge
- Fully responsive - try it on mobile!


TROUBLESHOOTING:
================
Q: Login not working?
A: Make sure you're using the correct email and password

Q: Data not saving?
A: Check if your browser allows localStorage

Q: Page not loading correctly?
A: Clear browser cache and try again

Q: Want to reset everything?
A: Clear browser data (localStorage) or use Incognito mode
