HOW TO ADD THE ERD IMAGE
========================

Please save your ERD image to this folder with the name "ERD.png" (or "ERD.jpg")

Steps:
1. Take a screenshot of your ERD diagram (the one with Patient, Doctor, Admin, and Appointments entities)
2. Save it as: ERD.png
3. Copy it to this docs folder:
   C:\Users\asser.fouad\OneDrive\Desktop\AF-MAIN\medical-appointment-system\docs\

4. Then run these commands:
   cd medical-appointment-system
   git add docs/ERD.png
   git commit -m "Add Entity Relationship Diagram (ERD)"
   git push origin main

5. Delete this instruction file after adding the ERD
