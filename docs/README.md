# Medical Appointment System - Documentation

This folder contains the project documentation and diagrams.

## Contents

- **ERD.png** - Entity Relationship Diagram showing the database structure
- **Documentation.pdf** - Complete project documentation including:
  - Team members
  - Problem statement
  - Objectives
  - Project features
  - Test cases

## Database Schema

The system uses the following entities:
- **Patient** - Patient information and credentials
- **Doctor** - Doctor profiles, specialties, and credentials
- **Admin** - System administrators
- **Appointments** - Appointment bookings linking patients and doctors
